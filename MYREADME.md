# DJS Portfolio Piece – Podcast App (React)

## Description 📜:

We were tasked to build upon our previous project(DJS05), where we were to build an advanced podcast browsing experience that allows users to dynamically search, sort, filter, and paginate a list of podcast shows. In addition, our podcast app was to navigate seemlessly between pages (with less to no lag, for better User Experience). With this final project, we were to add new features, to complete the Podcast App that we've been building component at a time.

## Features 🎨:

- Podcast cards.
- Within the cards: Podcast - title, seasons, genre and last updated date.
- Capabilities to: search, sort, filter, and paginate through Podcast information - to better customise for user experience (and preference).
- Search Functionality: Implement a flexible search that matches any part of the podcast title. Results should update dynamically as the user types or upon submission. Ensure that search results integrate with current filters, sorts, and pagination without resetting them.
- Sorting Options: Allow sorting podcasts by: Newest first (based on last updated date), Title A–Z and Z–A. Sorting must work in tandem with any search or filter criteria.
- Filter: Enable genre-based filtering using a dropdown or multi-select input. Ensure filters work alongside current search, sort, and pagination state. Maintain selected filters when navigating between pages or updating the list.
- Pagination: Display podcasts in manageable chunks using pagination, load-more, or infinite scroll. Ensure that pagination respects the currently active search, filter, and sort state. Keep all UI selections intact while navigating pages.
- Modular coding (JavaScript and CSS).
- Use of tools (Vite) and libraries (React, React-Router) to enhance viewing experience and coding efficiency.
- Theme toggle: for light and dark mode, with all styling to be done to enhance User Experience.
- Favourites Icon: the favourites icons (heart filled and heart empty) to save and unsave episodes, that will appear on their own Favourites Page.
- Favourites Page: the page will display all episodes in a list, under their show titles - in whatever sorting filter options picked. With date and time saved, to enhance filtering method. A "Clear All" button to clear out all your favourite episodes.
- Global Audio Player: with start and stop, volume options, audio listening bar to determine where your audio is and where you can shift it to, a stop button to exit the Audio Player. All with capabilities to view the player and listen throughout your destination in your Podcast App.
- Recommended Shows Carousel: multiple shows (Podcast cards) to be displayed from left to right, hover effect and clickability to enter for the shows details to still be diplayed. An automatic back to the start to shows, while automatically sliding (you can manually slide too).
- Metatags: for social media interaction and searchability of the Podcast App.
- Error and Loading warnings and messages, throughout interacting with the app.
- Back and Home buttons to return to your starting point or preferred starting point.
- API fetching of podcasts too.
- Responsiveness (changing screen viewing visuals per screen size): desktop, tablet, mobile.
